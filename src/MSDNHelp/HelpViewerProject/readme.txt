Help Viewer 2.0 example code Project 
  by Robert Chandler of www.Helpware.net in partnership with Paul O'Rear [MS] and the Microsoft Help team.

* Example code for accessing the Help API of both Visual Studio 11 and Windows 8 desktop.
* Code repository: http://helpviewer2.codeplex.com/
* Code docs: http://hv2.helpmvp.com/

----------------------------------------------------------------

HV2ApiExplore.sln -- Example 1 explores all aspects of the Help Viewer 2.0 API plus more.

HV2Viewer.sln -- Example 2 is a simple HV2 help Viewer.

HV2Everything.sln -- Contains both examples.
